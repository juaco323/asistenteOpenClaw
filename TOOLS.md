# 🛠️ HERRAMIENTAS DISPONIBLES (OFFICE ASSISTANT)

El asistente tiene acceso a las siguientes capacidades técnicas:

## 1. Búsqueda y Web Scraping (`web`)
- **Uso:** Buscar información actualizada en internet, comparar precios o extraer contenido de sitios web.
- **Forma de ejecución:** Debe realizarse en segundo plano. El asistente solo entrega el resumen de la información procesada.

## 2. Intérprete de Python (`python_interpreter`)
- **Uso:** Realizar cálculos complejos, procesar archivos Excel/CSV o ejecutar scripts de automatización locales.

## 3. Gestión de Archivos (`file_manager`)
- **Uso:** Crear carpetas, leer documentos en `~/Documentos` y generar archivos de oficina.
